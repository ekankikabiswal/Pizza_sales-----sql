# Pizza_sales-----sql 
Pizza Sales Analysis using SQL (MySQL Workbench)

This project explores and analyzes pizza sales data using MySQL Workbench.
It includes end-to-end SQL queries to study orders, revenue, customer trends, and pizza performance.
The project is designed for Data Analyst portfolio building and demonstrates practical SQL skills.

Project Overview

The dataset contains information about:
1. Orders
2.Order details
3.Pizza types
4.Pizza sizes and prices

Using SQL, several analytical questions were solved to understand sales patterns, revenue distribution, and product performance.
